# Student Management System API

## Features

- ASP.NET Core Web API
- CRUD Operations
- JWT Authentication
- Swagger Documentation
- Global Exception Middleware
- Built-in Logging
- DTO Validation
- Repository Pattern
- SQL Server

## Technologies Used

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- JWT Authentication

## Setup Steps

1. Clone repository
2. Open project in Visual Studio 2022
3. Update connection string in appsettings.json
4. Run migration
5. Run project
6. Open Swagger

## API Endpoints

### Authentication

POST /api/Auth/login

### Students

GET /api/Student

POST /api/Student

PUT /api/Student/{id}

DELETE /api/Student/{id}
